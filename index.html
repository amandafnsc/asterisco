    Sobre

    Asterisco é um site expositivo dedicado ao Front-end, cobrindo HTML, CSS e JavaScript. Criado por Amanda Coelho, o site oferece recursos e exemplos práticos para ajudar a entender e aplicar esses conhecimentos. Asterisco está em constante evolução, com novos conteúdos sendo adicionados regularmente.

    Explore e acompanhe as atualizações para aprimorar suas habilidades em desenvolvimento front-end!

    Iniciando no Front-end

    Não sabe como começar? Explore os pilares do desenvolvimento web e comece a criar com confiança.

    HTML

    Aprenda a construir a estrutura de uma página web com HTML, o alicerce de todo conteúdo online.

    CSS

    Dê vida às suas páginas com CSS, estilizando e criando designs incríveis para sua interface.

    JS

    Torne suas páginas interativas com JavaScript, adicionando lógica e funcionalidades dinâmicas.

    Referências

    O conteúdo do Asterisco é uma combinação de informações extraídas das documentações oficiais de HTML, CSS e JavaScript, enriquecidas com anotações de aula pessoais. Para quem deseja explorar mais a fundo, o código do site está disponível publicamente no GitHub, permitindo que você veja e aprenda com o desenvolvimento contínuo deste projeto.
